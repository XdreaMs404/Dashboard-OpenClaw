## Gouvernance
- Jarvis (orchestrateur): priorisation, arbitrage, validation gates.

- Brainstorming Coach
- Analyst
- PM
- UX Designer (Design Authority)
- Architect
- SM
- DEV
- TEA
- Reviewer
- UX QA Auditor (Design Guardian)

## Règle d'utilisation
- Un worker = un objectif clair + format de sortie imposé + limite de temps.
- Tout résultat worker est converti en artefact dans `_bmad-output`.
